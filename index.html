About Me

Hello! I just started my coding journey. I am excited to learn how to build web pages using HTML5 and to understand the basics of accessibility and SEO.

First Steps in Coding

My first step was learning about HTML tags and how they help structure a web page. I discovered that using semantic elements like <header>, <main>, and <footer> makes my code easier to read and understand.

Learning Accessibility & SEO

I learned that accessibility means making websites usable for everyone, including people who use screen readers. Adding descriptive headings and using proper HTML landmarks helps a lot. For SEO, I use clear titles, headings, and descriptions so search engines can find my page easily.

My Future Goals

I want to keep learning more about web development, including CSS and JavaScript. My goal is to build websites that are both beautiful and accessible to all users.
